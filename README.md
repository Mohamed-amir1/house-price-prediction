# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview
This project uses Machine Learning (Linear Regression) to predict house prices based on features such as area, bedrooms, bathrooms, and other property characteristics.

---

## 📊 Dataset
The dataset contains information about houses including:
- Area
- Bedrooms
- Bathrooms
- Parking
- Furnishing status
- Other features
